# Portal Digital de Agronomia – IFSertãoPE CPZR

## 📌 Finalidade do Site
Este site foi criado como um **ambiente digital para a coordenação de Agronomia**, com o objetivo de **disponibilizar e facilitar o acesso aos conteúdos do curso** do **IFSertãoPE – Campus Petrolina Zona Rural**. O portal reúne materiais de estudo, artigos, projetos acadêmicos e informações sobre eventos, proporcionando uma experiência organizada e acessível para alunos e professores.

---

## 💻 Tecnologias Utilizadas

O site foi desenvolvido utilizando as seguintes tecnologias:

- **[Hugo](https://gohugo.io/)** – Framework estático de sites, rápido e flexível.
- **[Docsy Theme](https://www.docsy.dev/)** – Tema Hugo para documentação e portais institucionais.
- **SCSS/SASS** – Para personalização de cores, tipografia e layout.
- **Netlify** – Plataforma de deploy contínuo para sites estáticos.
- **Font Awesome** – Biblioteca de ícones utilizada no site.
- **Git & GitHub** – Controle de versão e hospedagem do código-fonte.
